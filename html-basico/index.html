HTML Básico - Estrutura Fundamental

1. Estrutura Básica

Todo documento HTML deve ter esta estrutura básica:

  • <!DOCTYPE html> - Declara o tipo de documento
  • <html> - Elemento raiz
  • <head> - Metadados do documento
  • <body> - Conteúdo visível

2. Tags Essenciais no Head

  • <meta charset="UTF-8"> - Codificação de caracteres
  • <meta name="viewport"> - Responsividade
  • <title> - Título da página

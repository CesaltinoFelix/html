8. Boas Práticas

  • <table> - Container principal
  • <thead> - Cabeçalho da tabela
  • <tbody> - Corpo principal dos dados
  • <tfoot> - Rodapé (totais, resumos)
  • <th> - Células de cabeçalho
  • <td> - Células de dados
  • <caption> - Título/descrição da tabela
  • scope="col|row" - Melhora acessibilidade
  • colspan - Células que ocupam múltiplas colunas
  • rowspan - Células que ocupam múltiplas linhas
